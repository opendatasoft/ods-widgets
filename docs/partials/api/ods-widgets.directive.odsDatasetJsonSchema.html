odsDatasetJsonSchema
directive in module ods-widgets

Description

Generate a dataset's JSON schema and display it within a pre tag.

Usage

as element:
<ods-dataset-json-schema
       context="{DatasetContext}">
</ods-dataset-json-schema>

Directive info
