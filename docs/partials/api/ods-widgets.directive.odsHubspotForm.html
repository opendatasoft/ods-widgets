odsHubspotForm
directive in module ods-widgets

Description

Integrates a Hubspot form given a portal ID and the form ID.

Usage

as element:
<ods-hubspot-form
       portal-id="{string}"
       form-id="{string}">
</ods-hubspot-form>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
portalIdstring

The portal ID

formIdstring

The form ID

Example

<ods-hubspot-form portal-id="1234567" form-id="d1234564-987987987-4564654-7897-456465465"></ods-hubspot-form>
