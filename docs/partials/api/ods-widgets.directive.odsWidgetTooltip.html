odsWidgetTooltip
directive in module ods-widgets

Description

This directive is a helper for displaying custom tooltip. It allows to configure the usable fields in the tooltip and the template and does the html rendering giving back the compiled html to the calling widget. By default the template for the custom tooltip can access the record and a displayedFields array that lists the record fields that should appear in the tooltip.

Usage

as attribute
<ANY ods-widget-tooltip>
   ...
</ANY>
