odsThemePicto
directive in module ods-widgets

Description

This widget displays the "picto" of a theme, based on the themes setting in ODSWidgetsConfig. This element can be styled (height, width...), especially if the picto is vectorial (SVG).

Usage

as element:
<ods-theme-picto
       theme="{string}">
</ods-theme-picto>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
themestring

The label of the theme to display the picto of.
