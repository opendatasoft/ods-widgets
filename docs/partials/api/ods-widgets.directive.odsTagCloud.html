odsTagCloud
directive in module ods-widgets

Description

This widget displays a "tag cloud" of the values available in a facet (either the facet of a dataset, or a facet from the dataset catalog). The "weight" (size) of a tag depends on the number of occurences ("count") for this tag.

Usage

as element:
<ods-tag-cloud
       context="{CatalogContext|DatasetContext}"
       facet-name="{string}"
       max="{number}">
</ods-tag-cloud>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextCatalogContextDatasetContext

Catalog Context or Dataset Context to use

facetNamestring

Name of the facet to build the tag cloud from.

maxnumber

Maximum number of tags to show in the cloud.

(default: all)
