odsSocialButtons
directive in module ods-widgets

Description

This widget displays a share button that on hover will reveal social media sharing buttons.

Usage

as attribute
<ANY ods-social-buttons
     addthis-pubid="{string}"
     buttons="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
addthisPubidstring

Your AddThis account's public ID

buttonsstring

Comma separated list of buttons you want to display.

(default: 'google-plus,facebook,twitter')
