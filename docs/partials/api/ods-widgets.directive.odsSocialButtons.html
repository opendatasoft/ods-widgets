odsSocialButtons
directive in module ods-widgets

Description

This widget displays a series of buttons for easy sharing on social media.

Usage

as attribute
<ANY ods-social-buttons
     buttons="{string}"
     title="{string}"
     url="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
buttonsstring

Comma separated list of buttons you want to display.

(default: 'twitter,facebook,linkedin,email')

titlestring

Title of the post on social media

(default: current page's title)

urlstring

Url attached to the post on social media

(default: current page's url)
