refineOnClick
directive in module ods-widgets

Description

This directive will refine the given context(s) for a click on an element representing a record.

It works in conjunction with a finite set of other directives: odsCalendar odsMediaGallery odsMap odsChart * odsChartSerie

When clicking on an item, the contexts will be refined (using the values in the configured fields). By default, if you click on more than one items, the refinements will add up, which can be useful in situations with multiples values. If you'd prefer the refinement to be replaced each time you click, you can use refineOnClickReplaceRefine.

Usage

as attribute
<ANY refine-on-click>
   ...
</ANY>

Directive info

  • This directive creates new scope.
