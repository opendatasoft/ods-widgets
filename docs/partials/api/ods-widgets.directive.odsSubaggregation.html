odsSubaggregation
directive in module ods-widgets

Description

This widget computes aggregations on an analysis result. It It can be used with AngularJS's ngRepeat to simply build a table of analysis results.

Usage

as attribute
<ANY ods-subaggregation="{string}"
     ods-subaggregation-serie*="{number}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
odsSubaggregationstring

Analysis results

odsSubaggregationSerie*number

Aggregation expression
