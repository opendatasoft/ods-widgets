odsDatasetSchema
directive in module ods-widgets

Description

Display a table describing the schema of a dataset. For each field, it provides the label, name, description, type and an example

Usage

as element:
<ods-dataset-schema
       context="{DatasetContext}">
</ods-dataset-schema>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextDatasetContext

Dataset Context
