odsTimescale
directive in module ods-widgets

Description

Displays a control to select either:

  • last day

  • last week

  • last month

  • last year

Usage

as element:
<ods-timescale
       context="{DatasetContext|DatasetContext[]}"
       [time-field="{string}"]
       [*-time-field="{string}"]
       [default-value="{string}"]>
</ods-timescale>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextDatasetContextDatasetContext[]

Dataset Context or array of context to use

timeField
(optional)
string

Name of the field (date or datetime) to filter on

(default: first date/datetime field available)

*TimeField
(optional)
string

For each context you can set the name of the field (date or datetime) to filter on

(default: first date/datetime field available)

defaultValue
(optional)
string

Define the default timescale

(default: everything)
