odsDatetime
directive in module ods-widgets

Description

Get the ISO local datetime and store it into a variable (into the scope). Equivalent to moment().format() javascript call. The current scope gains a refreshDatetime method that will refresh the variable with the current datetime.

Usage

as attribute
<ANY ods-datetime>
   ...
</ANY>

Directive info

  • This directive creates new scope.
