odsLastReusesFeed
directive in module ods-widgets

Description

This widget displays the last 5 reuses published on a domain.

Usage

as element:
<ods-last-reuses-feed
       context="{CatalogContext}">
</ods-last-reuses-feed>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextCatalogContext

Catalog Context to use
