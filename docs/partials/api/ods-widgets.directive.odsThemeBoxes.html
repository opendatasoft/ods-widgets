odsThemeBoxes
directive in module ods-widgets

Description

This widget enumerates the themes available on the domain, by showing their pictos and the number of datasets they contain. They require the themes setting to be configured in ODSWidgetsConfig.

Usage

as element:
<ods-theme-boxes
       context="{CatalogContext}">
</ods-theme-boxes>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextCatalogContext

Catalog Context to pull the theme list from.
