moment
filter in module ods-widgets

Description

Render a given date in a specified format.

Usage

In HTML Template Binding

{{ moment_expression | moment:format }}

In JavaScript

$filter('moment')(date, format)

Parameters

ParamTypeDetails
datestringDateNumberArrayMoment

A date

formatstring

See http://momentjs.com/docs/#/displaying/format/ for the full list of options

Returns

string

The input date, formatted.
